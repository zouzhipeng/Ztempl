快速入手

Welcome

如何初始化？

只需要引用Ztempl.js后把需要绑定的html元素和绑定的对象通过Ztempl关联即可

  • Html:

    <span id=box></span>
  • Js:

    var Zdata = {};
    Ztempl(document.getElementById('box'),Zdata);

也可以把html元素换成JQ元素

  • Js:

    var Zdata = {};
    Ztempl($('#box'),Zdata);

还可以把html元素换成html字符串，并生成一个node，然后插入到任意一个你希望放置的元素里

  • Js:

    var Zdata = {title:'hello word'};
    var M = Ztempl('<span>{{title}}</span>',Zdata);
    $('#box').append(M.$nodes);

数据绑定

用{{}}标记绑定对象名称，{{}}标记内可支持三元，函数，自定义函数等写法，支持多变量。当绑定的是多变量，或不已变量开头，或被函数包裹，则变量也需要用$符号包裹以便识别

单变量绑定

  • 文本绑定
  • 属性绑定

文本绑定：把数据里的值自动输出至标签处

  • Html:

    <span id=box>{{title}}</span>
  • Js:

    var Zdata = {title:'hello word'};
    Ztempl(document.getElementById('box'),Zdata);

属性绑定：把数据里的值自动输出至元素属性里，img等引用资源的元素可在src属性前加下划线成_src来避免资源引用报错问题

  • Html:

    <div id='box'>
        <img _src='{{img}}'/>
    </div>
  • Js:

    var Zdata = {img:'src/img/img.jpg'};
    Ztempl(document.getElementById('box'),Zdata);

三元、函数等绑定方式

若标签不以变量开头，或标签内有多个变量，这需要用$包裹变量

  • Html:

    <span id=box>
        三元：{{sex==1?'male':'female'}};
        函数：{{new Date($birthday$).getFullYear()}};
    </span>
  • Js:

    var Zdata = {sex:1,birthday:964581138000};
    Ztempl(document.getElementById('box'),Zdata);

表单绑定

在表单元素的属性上加上Zbind属性用于绑定值，达到双向绑定，表单元素变更后变量直接变更，变量变更表单元素也一起变更

INPUT框绑定

input元素增加Zbind标签并设置值为绑定变量

  • Html:

    <div id=box>
        <input type="text" Zbind="title">
    <div/>
  • Js:

    var Zdata = {title:'hello word'};
    Ztempl(document.getElementById('box'),Zdata);

select框绑定

select元素增加Zbind标签并设置值为绑定变量,绑定的值会随着选项变更而变更

  • Html:

    <div id=box>
        <select Zbind="title">
            <option value="选项1">选项1</option>
            <option value="选项2">选项2</option>
        </select>
    <div/>
  • Js:

    var Zdata = {title:'选项1'};
    Ztempl(document.getElementById('box'),Zdata);

radio/checkbox框绑定(测试)

radio/checkbox元素增加Zbind标签并设置值为绑定变量,这两个元素必须设置name用来分组，设置value来设置选项值

checkbox多选时绑定的变量会成为用“,”拼接的字符串

  • Html:

    <div id=box>
        <label><input type="radio" Zbind="radio" name='interest' value='1'>单选1</label>
        <label><input type="radio" Zbind="radio" name='interest' value='2'>单选2</label>
        <label><input type="radio" Zbind="radio" name='interest' value='3'>单选3</label>
        <label><input type="radio" Zbind="radio" name='interest' value='4'>单选4</label>

        <label><input type="checkbox" Zbind="checkbox" name='interest' value='1'>多选1</label>
        <label><input type="checkbox" Zbind="checkbox" name='interest' value='2'>多选2</label>
        <label><input type="checkbox" Zbind="checkbox" name='interest' value='3'>多选3</label>
        <label><input type="checkbox" Zbind="checkbox" name='interest' value='4'>多选4</label>
    <div/>
  • Js:

    var Zdata = {radio:'',checkbox:''};
    Ztempl(document.getElementById('box'),Zdata);

循环绑定

在元素的属性上加上Zfor属性用于绑定循环元素，元素内的标签不再以外部数据为目标对象，而是以循环的每一项为目标对象

循环数组会产生索引Zkey,完整值Zvalue两个额外参数

可用Zdata附加额外参数(测试)

  • Html:


    <div id='box'>
      <ul>
        <li Zfor='arr_list' Zdata="{author:author}">{{Zkey}}[{{type}}]{{title}}--{{author}}</li>
      </ul>
    <div/>
  • Js:

    var Zdata = {
        arr_list:[
            {title:'标题1',type:'类型1'},
            {title:'标题2',type:'类型1'},
            {title:'标题3',type:'类型2'},
            {title:'标题4',type:'类型2'},
        ],
        author:'zzp'
    };
    Ztempl(document.getElementById('box'),Zdata);
